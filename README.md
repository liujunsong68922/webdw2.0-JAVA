# webdw2.0-JAVA
JAVA语言编写的WebDW2.0的客户端Demo程序。利用JAVA语言实现DataWindow的主要功能。

编译使用步骤如下：
1. 下载 webdw2.0-server 并打包发布，Tomcat监听80端口
2. git clone 下载 webdw2.0-JAVA 应用
3. mvn clean package ,程序打包
4. 在压缩包上点击右键，选择用jre打开
